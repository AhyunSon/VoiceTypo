# VoiceTypo

한국어 모음 인식 기반 프로젝트.

## 현재 상태 — main 재구축 중

`main` 은 재구축을 위해 비워둔 상태입니다.
이전 통합 프로젝트는 **`ahyun` 브랜치 `experiments/01_voicetypo_integrated/`** 에 보존돼 있습니다.

앞으로 `main` 은 회의에서 채택된 기능을 깨끗하게 재구현해 쌓아 올립니다.

## 협업 방법

- 각자 **개인 이름 브랜치**(`ahyun`, `jaewon`, `eunbin`, `jinhee`, `seoeun`, `sunmin`)에서 자유롭게 실험합니다.
- 실험은 각 브랜치의 `experiments/번호_주제/` 폴더로 관리하고, 폴더마다 README 를 둡니다.
- `main` 은 회의에서 채택된 기능을 `feature/` 브랜치로 재구현해 **PR 로만** 반영합니다.
- 개인 브랜치를 `main` 에 직접 merge 하지 않습니다.

## 브랜치

| 브랜치 | 용도 |
| --- | --- |
| `main` | 팀 통합본 (재구축 중) |
| 개인 브랜치 (6개) | 각자 실험 |
| `feature/*` | main 반영용 재구현 |

## 협업 가이드

자세한 규칙(브랜치 운영, 커밋 메시지, 실험 폴더 구조 등)은 팀 GitHub 협업 가이드를 따릅니다.
