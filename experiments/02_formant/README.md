# 02 포먼트 기반 7모음 인식

작성자: jaewon
브랜치: jaewon
기간:  2026-04-27 ~ 진행중

## 무엇 / 왜

포먼트(F1/F2/F3)만으로 한국어 7모음(아/에/이/오/우/으/어)을 실시간 인식.
학습된 표현(DNN) 대신 **연속적인 포먼트 값**이 필요(실시간 모핑용)해서,
포먼트 접근만으로 어디까지 갈 수 있는지를 끝까지 파고든 실험 라인.

핵심 질문: **포먼트 단독으로 다화자 환경 90% 인식이 가능한가?**

## 한눈에 — 핵심 결과

| 단계 | 방식 | 정확도 | 측정 대상 |
| --- | --- | --- | --- |
| 이전 시스템 | 앙상블+Kalman+wav2vec K-NN+F0 scale | 22.9% | 본인 1인 오프라인 |
| cleanup 후 baseline | Praat Burg 단독 + Bark Mahalanobis | 54.3% (19/35) | 본인 1인 오프라인 |
| Phase A (cal-free) | + VTLN 정규화 | 천장 54.3% | 본인 + 합성 다화자 |
| 캘리브레이션 라인 | cal + vote + GMM + bandwidth | 85.7% | 본인 1인, take-split |
| Phase B (학습) | Lobanov + LDA + 시간동역학 | 92.9% | 합성 다화자 / CV |
| 라이브 다화자 | cal + GMM 실사용 | 미흡 | 실제 다인 마이크 |

→ **결론:** cal-free 휴리스틱은 54.3%가 천장. 학습 기반(Phase B)은 합성 데이터에서 92.9%까지 도달 —
포먼트 단독 90%의 가능성은 보였으나, **실제 다화자 라이브 검증은 아직 미통과**.

## 접근 방향의 변화 (방향 전환 5회)

한 방법을 끝까지 민 게 아니라, **막힐 때마다 방향을 바꾼 기록**입니다.
수치보다 "왜 그 방향으로 틀었는가"가 이 실험의 핵심입니다.

전체 흐름: **22.9% → 54.3% → (천장)54.3% → 85.7% → 92.9% → 라이브 검증**

| 그때의 접근 | 정확도 | 막힌 점 → 다음 방향 |
| --- | --- | --- |
| 복잡한 앙상블 시스템 (앙상블·Kalman·wav2vec K-NN·F0 scale) | 22.9% | 모듈끼리 충돌 → ① 전부 걷어냄 |
| 단순 baseline (Praat Burg + Bark Mahalanobis) | 54.3% | 더 이상 안 오름 → ② cal-free 정규화 시도 |
| Phase A — cal-free 정규화 (VTLN 채택 / multi-proto·F3-ratio 폐기) | 천장 54.3% | cal 없이는 천장 → ③ 캘리브레이션 도입 |
| 캘리브레이션 라인 v1~v5 (cal+vote+GMM+bandwidth) | 85.7% | 휴리스틱 매칭 정체 → ④ 학습 기반으로 |
| Phase B — Lobanov 정규화 + LDA 학습 | 92.9% | 합성 데이터뿐 → ⑤ 실제 검증 필요 |
| 실제 다화자 라이브 검증 | 진행중 | ← 현재 여기 |

①~⑤ 가 다섯 번의 방향 전환. 각 전환의 계기는 위 표의 "막힌 점" 열 참고.

## 정확도 측정 기준

모든 수치는 **Top-1 정확도** = (예측 모음이 정답과 일치한 wav 수) ÷ (평가 wav 총수).

- **데이터:** 한국어 단모음 7개 × 5회 녹음 = 1인당 35 wav (자음 없는 단독 모음, 회당 약 1.5초).
- **측정 시점:** 각 wav 의 안정 구간(중앙 부근)에서 F1/F2/F3 추출 후 분류.

단계별 학습/평가 split 이 다르므로, 수치 비교 시 아래를 함께 봐야 함:

| 단계 | 학습 / cal set | 평가 set | 화자 |
| --- | --- | --- | --- |
| baseline | 없음 (학계 _REFS 휴리스틱) | 35 wav 전체 | 본인 1인 |
| 캘리브레이션 v2~v4 | takes 1~3 (21 wav) | takes 4~5 (14 wav) | 본인 1인 |
| 캘리브레이션 v5 | 5-fold leave-one-take-out | fold 당 7 wav, 누적 35 예측 | 본인 1인 |
| Phase B | within-speaker 5-fold CV / 합성 LOSO | — | 본인 + 합성 화자 |

**해석 시 주의 (중요):**
- 대부분 **본인 1인 오프라인** 측정 → 화자 독립성을 보장하지 않음.
- "합성 다화자"는 본인 포먼트에 성도 길이 배율(가상 남성 ×0.83, 가상 아동 ×1.20)을
  곱해 만든 가상 데이터 — **실제 다른 사람 목소리가 아님**.
- 오프라인 54.3% 와 실시간 환경 추정치(~37%) 사이에 큰 격차 — 실시간 청크 슬라이딩·VAD·EMA 가
  더해지면 정확도가 떨어짐.
- 따라서 위 표 수치는 **단계 간 상대 비교용**이며, 실사용 정확도 보장값이 아님.

## 실험 로그 (최신 → 과거, 실패도 기록)

### Phase B — 학습 기반 정규화 (2026-05~)
- Lobanov z-score 정규화 + LDA 학습 분류기 + 시간동역학(formant slope) → **92.9%**
  (본인 within-speaker 5-fold CV / 합성 다화자 LOSO 기준)
- 휴리스틱 거리매칭(Mahalanobis) 대신 데이터에서 결정경계를 학습하니 천장이 크게 오름
- 한계: 합성 다화자 데이터라 **실제 다화자 검증 필요** (상세: `evaluation/phase_b_v1_lobanov_lda.py`)

### 라이브 다화자 테스트 (2026-05-06)
- 실제 여러 사람 마이크 테스트 → cal + GMM 조합도 미흡
- 포먼트 추출 자체의 화자간 변동 + 어/오/우 후설군 중첩이 실사용에서 그대로 드러남

### Phase A — cal-free 정규화 (2026-04-29 ~)
- A1 VTLN: 가상 남성 48.6% → 54.3% 회복(+5.7%p). 본인(canonical) 화자엔 no-op
- A2 multi-prototype: 효과 미미 → **폐기**
- A3 F3-ratio(F1/F3, F2/F3): 효과 부정/미미 → **폐기**

### 캘리브레이션 라인 v1 ~ v5 (2026-04 ~ 2026-05)

## 주요 혼동 패턴

오답이 무작위가 아니라 특정 모음 쌍에 집중됨 — 진단으로 확인한 패턴:

| 혼동 | 원인 |
| --- | --- |
| 오 ↔ 우 | 둘 다 후설 원순 모음. F1·F2 가 가깝고 화자별 입술 둥글기 차이로 겹침 |
| 어 → 으 | 본인 "어" 가 bimodal 발음 (열린/닫힌 두 모드) — 닫힌 모드가 "으" 영역으로 |
| 에 ↔ 이 | 전설 모음 군집. F2 가 모두 높아 변별폭이 좁음 |
| 고 F0 화자 전반 | F1 이 F0 의 harmonic 으로 오인 추출 (여성·아동에서 빈발) |

→ 어/오/우 **후설군 중첩**이 가장 큰 벽. F1-F2 공간에서 화자간 변동이
모음간 거리보다 커서, 어떤 정규화로도 완전 분리는 불가능 (음성학계 합의).

## 핵심 인사이트

1. **복잡도가 정확도를 깎고 있었다.** 앙상블·Kalman·wav2vec K-NN·F0 scale 을 모두
   걷어내자 22.9% → 54.3%. 단순한 baseline 이 복잡한 시스템을 이김.
2. **병목은 분류기가 아니라 포먼트 추출.** 추출값이 학계 대비 6/7 모음에서 ≥100Hz
   어긋남 — 분류기를 아무리 고쳐도 추출이 틀리면 천장이 명확.
3. **cal-free 휴리스틱엔 천장이 있다 (54.3%).** 학계 _REFS 거리매칭만으로는 더 못 감.
   90%를 노리려면 캘리브레이션 또는 학습이 필수.
4. **학습이 휴리스틱을 이긴다.** LDA 가 데이터에서 결정경계를 직접 배우니 92.9%.
   단, 학습 데이터가 필요하다는 비용이 생김.
5. **단일 화자·합성·오프라인 수치는 실사용을 보장하지 않는다.** 라이브 다화자
   테스트에서 그대로 무너짐 — 측정 환경을 명시하지 않은 정확도는 의미가 약함.

## 남은 과제

1. **실제 다화자 검증** — Phase B 92.9%는 합성 데이터 결과. 실제 여러 화자(연령·성별 다양)
   녹음으로 재측정 필요. 검증 계획·필요 화자 규모는 `evaluation/speaker_independence_plan.md` (6~26명).
2. **오프라인 ↔ 실시간 격차** — 오프라인 54.3% vs 실시간 추정 ~37%. 실시간 파이프라인
   (청크 길이, VAD, EMA 안정화)에서 정확도가 깎이는 원인 규명·보정 필요.
3. **포먼트 추출 정확도** — 본인 데이터에서도 6/7 모음이 학계 대비 ≥100Hz 어긋남.
   특히 고 F0 화자(여성/아동)의 F1 이 F0 harmonic 으로 오인됨 — F0-aware 제약을 검증했으나 아직 미통합.
4. **어/오/우 후설군 중첩** — F1-F2 공간에서 화자간 변동이 모음간 거리보다 큼.
   정규화로 완화는 되나 본질적 한계 (50년 음성학 연구 합의).
5. **_REFS 학계 출처 보강** — 현재 단일 출처(Yoon 2015 추정) 기반. 변희경 2018 등
   다중 출처 수치 미확보 (`evaluation/refs_validation.md`).
6. **Phase B 실시간 통합** — Lobanov+LDA 가 92.9%를 냈으나 아직 평가 스크립트 수준.
   main 실시간 파이프라인에 통합되지 않음.
