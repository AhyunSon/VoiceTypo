## 3. 측정 환경 비교 — 미디어 아트 시나리오 적합성

| 측정 환경 | 미디어 아트 적합성 | 이유 |
|---|---|---|
| **단독 모음 발화 (sustained)** | ★★★★★ | 작품에서 관객은 "아", "에" 등을 또렷하게 한 번 길게 발음하는 형태 (현재 데이터셋 녹음 프로토콜과 일치) |
| 문장 내 모음 (CV/VC) | ★★★ | 자음 조음 영향 (coarticulation) 으로 F2 변동 큼 |
| 자연발화 코퍼스 | ★★ | undershoot 발생 → 모음 공간 작아짐 |
| 아나운서 (8명 정도) | ★★★ | 명료 발화이지만 평균값이 일반 화자보다 표준에 가까움 (편향 가능) |
| 폭넓은 연령 일반인 (수십명) | ★★★★ | 관객 다양성 시뮬레이션. 미디어 아트의 진짜 분포에 근사. |

**권장 우선순위**:
1. **변희경 (2018)** — 41명 폭넓은 연령. 일반 관객 분포에 가장 근사.
2. **하영우·오재혁 (2017)** — 8명 아나운서. 명료 발화 평균.
3. Yoon (2015) — 현재 사용 중. 출처/표본 재확인 후 유지/교체 판단.
4. 김순옥·윤규철 (2015) — 자연발화. 본 작품 시나리오와 거리 있음. 보조 참고만.

## 4. 추천 _REFS 산출 방법 (사용자 결정 필요)

### 옵션 A: 단일 출처 사용 (변희경 2018)
- 장점: 가장 다양한 화자 모집단. 일관된 측정 프로토콜.
- 단점: 한 연구의 측정 오차에 종속.
- 절차: 변희경 2018 의 평균값과 표준편차 그대로 사용.

### 옵션 B: 두 출처 가중 평균 (변희경 2018 + 하영우·오재혁 2017)
- 장점: 측정 오차 평균화.
- 단점: 출처 간 측정 환경 차이 무시.
- 절차: 화자 수로 가중 평균 (변희경 41명, 하영우 8명 → 비율 ~5:1).

### 옵션 C: 메타분석식 가중 평균 (3 출처)
- 장점: 가장 견고. 출처 간 분산을 σ 산출에 반영.
- 단점: 자연발화 코퍼스 포함 시 평균이 안쪽으로 끌려 들어감.
- 절차: 단독 모음 환경 출처(변희경 + 하영우)만 mean, 자연발화(김순옥)는 σ 보강용으로만.

**Claude 추천**: **옵션 A → 옵션 C 순으로 단계적 검증**.
1. 먼저 옵션 A 로 _REFS 교체 → baseline_simple 재측정.
2. 화자 독립성 측정 후, σ 가 너무 좁다고 판단되면 옵션 C 로 σ 만 보강.
3. mean 자체는 절대 본인 1인 데이터로 보정 금지.

## 5. σ (표준편차) 산출 정책

현재 σ = `(max - min) / 3` 가정. 이는 mean ± 1.5 SD = full range 가정 (정규분포 약 87% 커버).

학계 논문에서는 보통 σ 자체를 보고함. 다음 둘 중 하나:
- **σ 그대로 사용**: 학계 화자 풀 σ 그대로
- **σ 1.2~1.5x 확장**: 현장(미디어 아트) 화자가 학계 측정 화자보다 발음 일관성이 떨어진다고 가정 (관객은 음성학자가 아님)

**Claude 추천**: σ 1.3x 확장. 현장 발음 변동성 흡수용 마진.

예: 변희경 σ_F2 = 200 이면 코드에서 σ = 260 사용.

## 6. F3 처리

현재 _REFS 는 F3 도 포함 (오/우 구분용 보조).

학계 논문 다수가 F1/F2 만 보고하고 F3 는 빠진 경우 많음. **검증 필요**.

만약 F3 평균값을 학계에서 못 얻으면:
- 옵션: 현재 _REFS 의 F3 값 유지 (Yoon 출처 그대로)
- 옵션: F3 가중치(`_W_F3 = 0.30`) 를 0 으로 두고 F3 사용 안 함

baseline_simple.py 의 결과를 보면 F3 가 결정적 변별 요소가 아닌 모음이 많음. F3 우선순위 낮춰도 큰 손실 없을 가능성.
