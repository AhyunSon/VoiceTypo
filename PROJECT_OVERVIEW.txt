VoiceTypo Integrated - 프로젝트 개요
=====================================
작성일: 2026-03-05

목표:
  voice_pitch_detector-main/index.html 의 기능을
  Python(PySide6) 앱으로 대체하고, SDF 텍스트 모핑을 통합

원본 HTML 기능:
  - 마이크 실시간 입력
  - VAD (Voice Activity Detection)
  - 피치 감지 (YIN 알고리즘)
  - 모음 인식 (MFCC 기반, 8개 한국어 단모음)
  - 비브라토 감지
  - 시각 효과 (피치->스케일/색상, 볼륨->크기, 비브라토->떨림)
  - 캘리브레이션 (기준음, 모음별)
  - 디버그 패널 (스펙트럼, 각종 수치)

최종 데이터 흐름:
  마이크 (sounddevice)
    |
    +-> pitch_detection/  --> 주파수(Hz)
    +-> vowel_recognition/ --> 인식된 모음 (아/어/오/우/으/이/에)
    |
    +-> ui/main_window.py
          |
          +- text_effects/   : 피치->스케일, 볼륨->크기, 비브라토->떨림
          +- text_morphing/  : 모음 변경 시 SDF 모핑 애니메이션

폴더 구조:
  VoiceTypo_integrated/
  +-- main.py                      # 진입점 (모듈 통합)
  +-- requirements.txt
  |
  +-- audio_capture/               # 마이크 입력
  +-- pitch_detection/             # 피치 감지 (YIN)
  +-- vowel_recognition/           # 모음 인식 (여러 방식 비교 테스트)
  |   +-- method_1_mfcc_cosine/    # 원본 HTML 방식 포팅
  |   +-- method_2_formant_lpc/    # 포먼트 F1-F2 기반
  |   +-- method_3_mfcc_cmvn_svm/  # MFCC+CMVN+SVM (추천)
  |   +-- method_4_wav2vec2/       # wav2vec2 음소 인식
  |   +-- method_5_whisper/        # Whisper 기반
  |
  +-- text_morphing/               # SDF 글리프 모핑 (이미 구현됨)
  +-- text_effects/                # 시각 효과 (스케일, 색상, 떨림)
  +-- ui/                          # PySide6 메인 윈도우
